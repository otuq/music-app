# Music App Portfolio

Next.js App Router を使用したフルスタック投稿アプリです。

## Demo

https://xxxxx.vercel.app

## 使用技術

- Next.js
- React
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- Neon
- Auth.js
- Vercel

## 主な機能

- GitHubログイン
- 投稿作成
- 投稿一覧表示
- 投稿編集
- 投稿削除
- ユーザーごとの投稿管理
- PostgreSQLへのデータ保存
- Vercelデプロイ

## 学習・実装ポイント

- Server Components と Client Components の切り分け
- Server Actions によるDB操作
- Auth.js によるOAuth認証
- Prismaによる型安全なDB操作
- Neon PostgreSQLとの接続
- Vercelへの本番デプロイ
